# wydarzenia

Aplikacja ASP.NET Core (frontend + Web API) do zarządzania wydarzeniami w Warszawie.

## Uruchomienie

    dotnet run --project Wydarzenia.Api/Wydarzenia.Api.csproj

Po uruchomieniu otwórz adres podany w konsoli (np. `https://localhost:xxxx` lub `http://localhost:xxxx`).

## Funkcje

- lista wydarzeń,
- pełny CRUD przez `api/events`,
- responsywny interfejs,
- przełącznik trybu jasny/ciemny.
